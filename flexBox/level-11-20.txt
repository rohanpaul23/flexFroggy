#LEVEL 15

Flex items have a default order value of 0, 
therefore items with an integer value greater than 0 will be displayed after any items that have not been given an explicit order value.

Negative values are also supported with order. 
If you want to make one item display first and leave the order of all other items unchanged, you can give that item order of -1. 
As this is lower than 0 the item will always be displayed first.

order : -1;
